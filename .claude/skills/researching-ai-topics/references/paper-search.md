# Paper Search Guide

## Search Sources

| Source | URL | Best For |
|--------|-----|----------|
| arXiv | arxiv.org | Latest preprints, ML/AI |
| Semantic Scholar | semanticscholar.org | Citation analysis, related papers |
| Google Scholar | scholar.google.com | Broad academic search |
| Papers With Code | paperswithcode.com | Papers with implementations |
| Hugging Face Papers | huggingface.co/papers | Daily curated ML papers |

## Search Strategy

1. **Start broad**: search key terms on Semantic Scholar/arXiv
2. **Filter recent**: focus on last 12-24 months for fast-moving topics
3. **Check citations**: high-citation papers are influential
4. **Find implementations**: Papers With Code links code to papers
5. **Read abstracts first**: filter before deep reading

## Paper Summary Template

```markdown
### [Paper Title] (Year)
**Authors**: [names]
**Link**: [URL]
**Key contribution**: [1-2 sentences]
**Relevance**: [why it matters for this research]
**Limitations**: [what it doesn't address]
```

## Keyword Tips
- Use specific terms: "retrieval augmented generation" not "RAG"
- Add domain: "medical question answering" not just "QA"
- Include method: "multi-agent reinforcement learning"
